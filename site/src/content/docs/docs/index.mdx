---
title: repld documentation
description: A live Python kernel for your agent — shared namespace, channel push, browser automation, gists.
---

repld gives your agent a long-running Python kernel in your project directory:
shared `__main__` namespace between you and the agent, push notifications
instead of polling, a real browser as an object, and gists — integration code
that hardens into tools.
